Посилання на репозиторій:




Використані теґи HTML:

    <form></form> - теґ для фоми
   <legend></legend> - теґ для заголовку форми
    <label></label> - теґ для позначки(підпису) поля для заповнення
    <input></input>  - теґ для поля введення, який має різні типи:
   - <text> - тип для введення тексту
   - <email>  - тип для введення електронної пошти
  - <password> - тип для паролю
  - <checkbox> - тип для галочки
  - <radio> - тип для галочки, яку можна обрати лише одну з групи
   <select></select> - теґ для випадаючого списку
    <datalist></datalist>  - теґ для поля для введення, але з елементами випадаючого списку
    <textarea></textarea> - теґ для введення великого тексту 
   <meter> </meter> - теґ для шкали
