Використані теґи HTML:

    <h1></h1> - теґ для найбільшого заголовку
   <h2></h2> - теґ для другого найбільшого заголовку
    <h3></h3> - теґ для третього за розміром заголовку
    <p></p>  - теґ для абзацу
    <dfn></dfn> - теґ для визначення
    <hr>  - теґ для горизонтальної лінії
   <em></em> - теґ для курсиву
   <abbr></abbr> - теґ для абривіатури
   <br> - теґ для переходу на наступний рядок
   <ul></ul> - теґ для списку
    <li></li>  - теґ для елементу списку
    <img> - теґ для зображення
   <q> </q> - теґ для цитати
   <a> </a> - теґ для посилання
    <del></del> - теґ для закресленого тексту
   <ins></ins> - теґ для підкресленого тексту
    <span style="color: aqua;"></span> - теґ для тексту з атрибутом на блакитний
   колір
    <footer style="bottom: 0;width: 100%;position: fixed;"></footer>  - теґ для
   нижньої
       частини сторінки
    <pre></pre>  - теґ для тексту, який має зберігати відступи та переходи на
   наступний
       рядок

CSS:

width - атрибут ширини height - атрибут висоти background-color - атрибут кольору фону content - атрибут для списку, позначає підпис color - атрибут кольору тексту list-style - атрибут для списку, який визначає тип списку та вигляд маркера для кожного елемента border - атрибут для визначення границі елемента border-top border-left border-right border-bottom - атрибут для задання границі для різних сторін border-radius - атрибут для задання заокругленості границі елемента margin - атрибут для задання зовнішнього відступу margin-top margin-left margin-right margin-bottom - атрибут для задання зовнішнього відступу для різних сторін padding - атрибут для задання внутрішнього відступу text-decoration - атрибут для декорації тексту (підкреслення, закреслення ітд.) font-size - атрибут для задання розміру шрифта font-weight - атрибут для задання жирності тесту
